{{% blocks/lead color="primary" %}}
OllamAssist is a JetBrains IDE plugin that seamlessly integrates Ollama-powered AI capabilities directly into your development workflow. Chat with AI, leverage RAG for workspace context, get smart code completions, and generate commit messages—all locally and offline.

**Privacy First • Offline Ready • Fully Customizable**
{{% /blocks/lead %}}

{{< blocks/section color="dark" >}}
{{% blocks/feature icon="fas fa-comments" title="In-IDE Chat" %}}
Interact with Ollama models directly in your IDE. Get code explanations, discuss architecture, and receive context-aware suggestions powered by RAG (Retrieval-Augmented Generation).
{{% /blocks/feature %}}

{{% blocks/feature icon="fas fa-search" title="RAG Integration" %}}
Workspace-aware responses using your project's indexed sources. The AI understands your codebase and provides tailored solutions based on your context.
{{% /blocks/feature %}}

{{% blocks/feature icon="fas fa-keyboard" title="Smart Autocomplete" %}}
Press Shift+Space to get intelligent code suggestions. Enter to accept, any other key to dismiss. Experimental but powerful.
{{% /blocks/feature %}}

{{% blocks/feature icon="fas fa-git-alt" title="Commit Messages" %}}
Let OllamAssist generate conventional commit messages based on your git diff. Follow best practices automatically.
{{% /blocks/feature %}}

{{% blocks/feature icon="fas fa-wifi" title="Completely Offline" %}}
All processing is done locally. No data leaves your machine. Enjoy full functionality without an internet connection.
{{% /blocks/feature %}}

{{% blocks/feature icon="fas fa-cog" title="Highly Customizable" %}}
Configure separate Ollama instances for chat, completion, and embeddings. Customize system prompts. Adjust UI fonts and more.
{{% /blocks/feature %}}

{{< /blocks/section >}}

{{< blocks/section >}}

{{% blocks/feature icon="fas fa-download" title="5-Minute Setup" %}}
Download Ollama, select a model, install the plugin, and start coding with AI assistance. Simple as that.
{{% /blocks/feature %}}

{{% blocks/feature icon="fas fa-lock" title="Privacy-Focused" %}}
No cloud dependency. Your code stays on your machine. Perfect for sensitive projects and enterprise environments.
{{% /blocks/feature %}}

{{% blocks/feature icon="fas fa-star" title="Always Free" %}}
Open-source and MIT licensed. No subscriptions. No premium features locked behind paywalls.
{{% /blocks/feature %}}
